Classification Project

Project Overview

This project is a Machine Learning Classification System developed using Python. The model is trained on a dataset to classify data into different categories and predict the correct class for new inputs.

Features :

- Loads and processes the dataset
- Trains a Machine Learning model
- Makes predictions on new data
- Evaluates model performance
- Displays model accuracy

Technologies Used :

- Python
- Pandas
- Scikit-learn

Concepts Used :

- Data Preprocessing
- Train-Test Split
- Machine Learning Classification
- Model Training
- Model Prediction
- Accuracy Evaluation

Libraries Used :

- pandas
- scikit-learn

How to Run :

1. Open the project folder in VS Code.
2. Open the terminal.
3. Run the following command:

python classification.py

Sample Output :

Model trained successfully

First Prediction: 1
Actual Value: 1

Accuracy: 1.0
